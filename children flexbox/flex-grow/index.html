flex-grow property

This property works only when there is available space for the boxes to grow in the specified ratio. If the container covers the entire width of the screen then there is no need for the custom width specification.

If you want the container to cover the entire width of the screen.Set the .father container flex-direction property to column as used here.

Because in the flex-direction property set to row you have to specify custom width.
